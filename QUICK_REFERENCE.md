# Quick Reference Guide - New Features

## 🔗 Site-to-Site Event Linking

### When to Use
Event is "Set Down" + has pending items

### How to Access
Event Details → **Blue "Site-to-Site Link" button** → Wizard

### Simple Logic
- ✅ **Ticked** = Transfer to new site
- ☐ **Unticked** = Return to warehouse

### Steps
1. Click "Site-to-Site Link"
2. Fill new event details
3. Review items (all pre-ticked)
4. Untick unwanted items
5. Add warehouse items if needed
6. Click "Create Linked Event"

---

## 🗑️ Delete Events Safely

### Location
Events List → Actions Column → **Trash Icon**

### What Happens
- Event deleted permanently
- ALL items returned to warehouse automatically
- Activity logs created
- Transaction-safe (rollback on error)

### Confirmation
Shows exactly how many items will be freed

## 📸 Photo Features

### Full-Size Preview
**Where**: Event Details → Dispatch Photos section
**How**: Click any photo thumbnail
**Close**: Click outside, press ESC, or click X

### PDF Reports
- **Dispatch PDF**: 70×70px photos (green border)
- **Receive PDF**: 65×65px dual photos (green + red borders)
- **Side-by-Side**: Compare dispatch vs return condition

### Inventory Grid
Now shows uploaded primary photos instead of placeholders!

## 💡 Pro Tips

1. **Before Deleting**: Events can be deleted safely - items auto-return to warehouse
2. **Photo Inspection**: Click photos for full-size view before generating PDFs
3. **Site-to-Site Chains**: Can link multiple events (A→B→C→D)
4. **Primary Photos**: Upload and mark as primary to show in inventory grid
5. **PDF Photos**: Upload dispatch/return photos for best-looking reports

---

## 🚨 Important Notes

- **Duplicate Prevention**: Don't double-click submit buttons (auto-blocked)
- **Item Status**: Transferred items don't need to be "Available"
- **Deletion**: Cannot delete Active or Set Down events (must change status first)
- **Photos**: Max 4MB per photo (JPEG, PNG, WEBP)
